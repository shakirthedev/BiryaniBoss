# Biryani Boss 🍲

A fast-paced desi dhaba time management game. Serve biryani by tapping ingredients in the correct order before time runs out!

## 🎮 How to Play

1. **Check the Order:** Look at the customer's requested ingredients at the top of the screen.
2. **Tap Fast:** Tap the matching ingredient buttons in the exact sequence shown.
3. **Beat the Clock:** Complete the order before the timer bar runs out.
4. **Build Combos:** Get 3 correct taps in a row for a score multiplier!
5. **Don't Mess Up:** Wrong ingredients will cost you precious time and reset your combo.

## ✨ Features

* **Desi Dhaba Vibe:** Warm colors, emoji graphics, and a handmade feel.
* **Dynamic Audio:** Fully synthesized sound effects using the Web Audio API, including a custom Dholak-style background beat, metallic "Deg Tak Tak" sounds, and speech synthesis for milestones!
* **Progressive Difficulty:** Rounds get harder with more ingredients and less time.
* **Milestones:** Reach every 3rd round to become the "Awesome Biryani Boss"!
* **Mobile-First:** Designed to be played on phones with easy-to-tap buttons.

## 🛠️ Tech Stack

* React 18
* TypeScript
* Tailwind CSS
* Web Audio API & Web Speech API
